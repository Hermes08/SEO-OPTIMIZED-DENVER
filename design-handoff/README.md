# design-handoff/

Carpeta lista para adjuntar a Claude (modo diseño) o entregar a un diseñador.

- **`DESIGN_BRIEF.md`** — léelo primero. Reglas, tokens actuales, inventario de páginas/componentes y recomendaciones de mejora.
- **`source-snapshot/`** — copia curada de los archivos relevantes para diseño (estilos, componentes, páginas y datos como referencia). Sin `node_modules` ni `.next`.
- **`screenshots/`** — capturas del sitio en vivo (opcional; pídelas si no están).

Sitio en vivo: https://sunny-praline-e0f93b.netlify.app/
